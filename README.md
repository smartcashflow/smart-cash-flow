# Smart Cash Flow for Small Businesses

This is the frontend-only UI/UX prototype of Smart Cash Flow for Small Businesses.

The prototype uses:

- HTML
- CSS
- JavaScript
- Chart.js
- Browser Local Storage for temporary prototype data

There is no backend, database server, or build step in this version. It can be opened directly in a browser or published with GitHub Pages.

## Open Locally

Open `index.html` in your browser.

## GitHub Pages

1. Push this folder to a GitHub repository.
2. Open the repository settings.
3. Go to Pages.
4. Select the main branch and root folder.
5. Save and wait for GitHub Pages to publish the prototype.

## Full-Stack Backup

The previous Node.js, Express, and SQLite version is saved in:

```text
fullstack-backup/
```

The earlier frontend-only version is still saved in:

```text
backups/frontend-localstorage-version/
```
